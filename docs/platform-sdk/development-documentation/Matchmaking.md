
### Matchmaking

<br>

#### Overview

For now, we only support quick match mode between two users.

#### API Introduction

##### Quick Match Enqueue

```
HypMatchmaking()->QuickMatch_Enqueue(const char* pool)
```

Please refer to comments of our SDK headers for details.

##### Quick Match Cancel

```
HypMatchmaking()->QuickMatch_Cancel();
```

Please refer to comments of our SDK headers for details.
